# Academic Research & Technical Documentation

This repository showcases technical articles, academic papers, and comprehensive problem-set resolutions developed during my Telecommunications Engineering degree. 

The documents demonstrate advanced proficiency in **LaTeX** for scientific typesetting, complex mathematical formatting, and rigorous academic structuring.

## 📂 Featured Documents

### 1. Antireflective Coatings (Revestimentos Antirreflexo)
A detailed technical article and conceptual mind map exploring the physics, materials, and engineering applications of antireflective coatings.
* **Highlights:** Advanced document structuring, integration of technical figures, and theoretical physics explanations.
* **Format:** Full LaTeX source code and compiled PDF.

### 2. Applied Electromagnetism: Comprehensive Problem Set
A complete, step-by-step analytical resolution of an extensive 65-question problem set on applied electromagnetism.
* **Highlights:** Extensive use of complex mathematical environments, matrices, vector calculus, and differential equations formatting. Demonstrates both subject matter expertise and high-level technical documentation skills.
* **Format:** Full LaTeX source code and compiled PDF.

### 3. RF Transmission Lines Design (Stripline & Microstrip)
A comprehensive technical report detailing the analytical design and computational simulation of planar transmission lines operating at 12 GHz.

* **Highlights:** Calculation of characteristic impedance ($Z_0$), effective permittivity ($\epsilon_{eff}$), and phase delay. Includes computational sensitivity analysis (MATLAB) to validate a reflection coefficient of $|\Gamma| \le 0.05$ and delay tolerances across the operational bandwidth.
* **Format:** Full LaTeX source code and compiled PDF.
